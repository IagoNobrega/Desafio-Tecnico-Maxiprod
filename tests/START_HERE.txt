REGRA 1: ❌ Menor de Idade NÃO Pode Ter Receita
─────────────────────────────────────────────
  ✅ Pessoa < 18: não pode receita
  ✅ Pessoa ≥ 18: pode receita
  ✅ Qualquer idade: pode despesa
  ✅ Edge case: 18 exatamente
  Cobertura: 7 testes (Unit + Integration + E2E)

REGRA 2: 📂 Categoria Respeita Finalidade
─────────────────────────────────────────
  ✅ Categoria Receita = só receita
  ✅ Categoria Despesa = só despesa
  ✅ Categoria Ambas = receita OU despesa
  ✅ Rejeita incompatibilidade
  Cobertura: 10 testes (Unit + Integration + E2E)

REGRA 3: 🗑️ Exclusão em Cascata de Transações
──────────────────────────────────────────
  ✅ Delete pessoa = delete transações dela
  ✅ Não afeta outras pessoas
  ✅ Funcionamento atômico
  ✅ Edge cases: N transações
  Cobertura: 10 testes (Unit + Integration + E2E)
